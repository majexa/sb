Немного о Мудулях Разделов

Модуль раздела представляет из себя несколько классов разных видов. Это:
- DdoPage-класс для сайта
- DdoPage-класс для админки
- PageModule (Page Module Installer). Класс, устанавливающий модуль
- Pma (Page Module Action). Различные действия специфичные для модуля. Вызываются в различных частях системы
  и др...
  
Модули поддерживают наследование всей группы классов с помощью наследования класса PageModule.
Таким образом PageModule-класс определяет предка модуля. Если из группы классов модуля, какой-то класс
будет отсутствовать, вместо него будет получен класс предка.
